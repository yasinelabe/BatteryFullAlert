Overview

Battery Full Alert is a lightweight desktop application designed to monitor your laptop's battery status and notify you when it reaches a user-defined level.

Features

  • Real-time battery monitoring with a visual chart.
  • Customizable alert percentage (e.g., notify at 90%).
  • Sound alerts with adjustable volume control.
  • Supports custom sound files for alerts.
  • Minimizes to the system tray for uninterrupted monitoring.

Installation

Follow these steps to install and run the application:

  1. Download the latest release from our GitHub repository.
  2. Extract the downloaded file to your desired location.
  3. Run the executable file (BatteryAlertApp.exe) to start the application.
  4. Customize your settings in the "Settings" tab.

For developers, you can build the app from source using Python:

pip install -r requirements.txt
python main.py
